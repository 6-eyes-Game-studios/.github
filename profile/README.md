## 🎮 About Us

**6 Eyes Game Studios** is an independent game development studio crafting **"Trios: 734"** - a dark sci-fi action experience where biology became a weapon, humans are a relic, and morality is a personal choice.

Born from a single question: *What does it mean to be created for someone else's purpose and learn to live by your own?*

---

## 🌌 Current Project: Trios: 734

Wake up on **Trios** - a moon of a gas giant called **Erebus**. Humanity vanished 120 years ago. The creatures they created now rule the ruins.

You are **Evo** - a prototype with an open-ended genetic architecture. The only one of your kind capable of reshaping your own body in real time.

| Feature | Description |
|:--------|:------------|
| **Adaptive Evolution** | Transform between 4 unique forms (Prototype, Unstable, Perfect, Whole) |
| **Deep Narrative** | Uncover 3 layers of truth about Earth's silence |
| **Living World** | Factions, cults, and creatures with their own cultures |
| **Polymer Combat** | Build spell chains from absorbed DNA |
| **20-30 Hours** | Single-player experience with multiple endings |

## 🤝 Join the Development

This project is **100% open source** under **GPLv3**. We welcome contributors of all skill levels!

**We're looking for:**
- `🖥️ Programmers` - Unity, C#, gameplay systems
- `🎨 3D Artists` - Characters, environments, props
- `📖 Narrative Designers` - Lore writing, quest design
- `🎵 Sound Designers` - Music, ambience, SFX
- `🐞 Playtesters` - Bug hunting and feedback

**How to start:**
1. Join our [Discord](https://discord.gg/GEyVXwjMuV)
2. Look for issues with `good first issue` label
3. Read `CONTRIBUTING.md` for guidelines

**Found a bug?** You can:
- Create an issue directly on GitHub
- Report it on our Discord server
